# 👻 pacman.py
 
Juego de Pac-Man construido con Python y la librería Turtle. El jugador intenta comer todos los puntos del tablero y evita ser atrapado por los fantasmas.
 
---
 
## 📋 Cambios por commit
 
### Commit 1 — Fantasmas más rápidos
Primer commit, donde se aumentó la velocidad de los fantasmas de 5 a 10, para que se muevan más rápido y poder aumentar la dificultad del juego.
 
### Commit 2 — Nuevo tablero
Segundo commit, donde se modificó la variable tiles para rediseñar el tablero, cambiando de lugar paredes y pasillos.
 
### Commit 3 — Método Manhattan
Tercer commit, donde se reemplazó choice(options) por options.sort() usando distancia Manhattan para que los fantasmas, al bloquearse, elijan la dirección que los acerque más a Pac-Man. Ahora los fantasmas persiguen activamente al jugador en lugar de moverse al azar.
